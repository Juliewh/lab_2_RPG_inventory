# Лабораторная работа №2 C#

Разработана система управления инвентарем для ролевой игры.

Система выполняет следующее:

1. Позволяет игрокам добавлять различные типы предметов в инвентарь, такие как оружие, броня, зелья и квестовые предметы.
2. Учитывает уникальные свойства каждого предмета, например, урон для оружия или защиту для брони.
3. Предоставляет возможность использовать или экипировать предметы, а также отображать информацию о текущем состоянии инвентаря.
4. Обеспечена возможность комбинирования или улучшения предметов.
5. Логика покрыта unit тестами
6. CLI нет
7. Соблюдено по каждой букве из SOLID в различных модулях проекта
